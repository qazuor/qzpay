# Instalación

QZPay se distribuye como múltiples paquetes. Instala solo lo que necesites.

## Paquete Core

El paquete core siempre es requerido:

<Tabs>
  <TabItem label="pnpm">
    ```bash
    pnpm add @qazuor/qzpay-core
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npm install @qazuor/qzpay-core
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @qazuor/qzpay-core
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash
    bun add @qazuor/qzpay-core
    ```
  </TabItem>
</Tabs>

## Configuración Típica

Para una configuración completa de Stripe + Hono:

```bash
pnpm add @qazuor/qzpay-core @qazuor/qzpay-stripe @qazuor/qzpay-drizzle @qazuor/qzpay-hono stripe drizzle-orm postgres hono
```

<Aside type="note">
Todos los paquetes de QZPay son solo ESM y requieren Node.js 22+.
</Aside>

## Dependencias Peer

Algunos paquetes tienen dependencias peer que debes instalar:

| Paquete | Dependencias Peer |
|---------|-------------------|
| `@qazuor/qzpay-stripe` | `stripe` |
| `@qazuor/qzpay-mercadopago` | `mercadopago` |
| `@qazuor/qzpay-drizzle` | `drizzle-orm`, `postgres` |
| `@qazuor/qzpay-hono` | `hono`, `zod` |
| `@qazuor/qzpay-nestjs` | `@nestjs/common`, `@nestjs/core` |
| `@qazuor/qzpay-react` | `react`, `react-dom` |
