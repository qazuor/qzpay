## Reportar Errores

Al reportar un error, por favor incluye:

### Información del Entorno

```
QZPay Version: x.x.x
Node.js Version: x.x.x
Operating System: Linux/macOS/Windows
Payment Provider: Stripe/MercadoPago
Database: PostgreSQL/SQLite/MySQL
```

### Pasos de Reproducción

1. Qué estabas tratando de hacer
2. Qué esperabas que sucediera
3. Qué sucedió realmente

### Reproducción Mínima

Si es posible, proporciona un ejemplo de código mínimo que reproduzca el problema:

```typescript
import { createBilling } from '@qazuor/qzpay-core';
import { createQZPayStripeAdapter } from '@qazuor/qzpay-stripe';

const billing = createBilling({
  paymentAdapter: createQZPayStripeAdapter({ secretKey: 'sk_test_xxx' }),
  storage
});

// Code that causes the issue
const result = await billing.customers.create({
  email: 'test@example.com'
});
// Error: ...
```

### Mensajes de Error

Incluye el mensaje de error completo y el stack trace.

## Solicitudes de Funcionalidades

Al solicitar una funcionalidad:

1. **Describe el problema** - ¿Qué problema estás tratando de resolver?
2. **Propón una solución** - ¿Cómo debería funcionar?
3. **Proporciona contexto** - ¿Por qué es importante para tu caso de uso?
4. **Considera alternativas** - ¿Qué otros enfoques has considerado?

## Problemas de Seguridad

<Aside type="caution">
**No reportes vulnerabilidades de seguridad públicamente.** Por favor envía los problemas de seguridad directamente a security@qazuor.com.
</Aside>

Incluye:
- Descripción de la vulnerabilidad
- Pasos para reproducir
- Impacto potencial
- Solución sugerida (si la hay)

Confirmaremos la recepción dentro de 48 horas y proporcionaremos actualizaciones sobre nuestra investigación.

## Mejores Prácticas de Soporte

Para obtener la mejor ayuda:

1. **Busca primero** - Verifica si tu pregunta ya ha sido respondida
2. **Sé específico** - Proporciona información detallada sobre tu problema
3. **Muestra tu trabajo** - Incluye código relevante y mensajes de error
4. **Ten paciencia** - Los mantenedores a menudo son voluntarios
5. **Contribuye** - Ayuda a otros cuando puedas
