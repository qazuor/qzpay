# Installation

QZPay is distributed as multiple packages. Install only what you need.

## Core Package

The core package is always required:

<Tabs>
  <TabItem label="pnpm">
    ```bash
    pnpm add @qazuor/qzpay-core
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npm install @qazuor/qzpay-core
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @qazuor/qzpay-core
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash
    bun add @qazuor/qzpay-core
    ```
  </TabItem>
</Tabs>

## Typical Setup

For a complete Stripe + Hono setup:

```bash
pnpm add @qazuor/qzpay-core @qazuor/qzpay-stripe @qazuor/qzpay-drizzle @qazuor/qzpay-hono stripe drizzle-orm postgres hono
```

<Aside type="note">
All QZPay packages are ESM-only and require Node.js 22+.
</Aside>

## Peer Dependencies

Some packages have peer dependencies you need to install:

| Package | Peer Dependencies |
|---------|-------------------|
| `@qazuor/qzpay-stripe` | `stripe` |
| `@qazuor/qzpay-mercadopago` | `mercadopago` |
| `@qazuor/qzpay-drizzle` | `drizzle-orm`, `postgres` |
| `@qazuor/qzpay-hono` | `hono`, `zod` |
| `@qazuor/qzpay-nestjs` | `@nestjs/common`, `@nestjs/core` |
| `@qazuor/qzpay-react` | `react`, `react-dom` |
